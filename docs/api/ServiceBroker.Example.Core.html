Namespace ServiceBroker.Example.Core

Interfaces

ICache

Represents an instance of a cache framework. The interface is an abstraction exposing only the subset of features from a caching framework needed by the implemented services of the example.

ICachedService

A called service wrapper which saves the values returned from the called service in the cache, such that consecutive calls within the same cache region reads data from the cache rather than call the external service.

IDynamicService

A called service wrapper, where the called service should be called every time wrapper is called.

IHttpClientWrapper

A service for calling external HTTP-resources.

IServiceBase

A base interface used for all external service types. Exposes the methods that must be implemented by all external service types.

IServiceBrokerService

A service broker wrapping calls to multiple external services, running them in parallel, handling timeouts for the service calls, and returning a combined result.

IServiceRepository

This interface represents a repository tha tis able to load data about one or more external services and/or tokens from a data source.

IStaticService

A service that always returns the same answer no matter which user the service is called for

ITaskScheduler

An interface that represents any service that is able to schedule an action on a background thread.

ITokenService

This interface represents a service that is able to evaluate one or more tokens on a text input or on the stored value of another token.
